# Bryson-Portfoilo

A high-performance, aesthetically pleasing Software Engineer portfolio built with **React**, **TypeScript**, and **Vite**.  
Designed to showcase engineering skills through a "Corporate Premium" aesthetic, featuring a custom dark/light theme system, interactive 3D visuals, and a responsive layout.

## üöÄ Features

-   **Premium Aesthetic**: Clean, corporate design with a focus on typography and whitespace.
-   **Dark & Light Mode**: Fully themed UI with a smooth toggle switch.
-   **Interactive Visuals**: Custom CSS-only "Code Editor" with syntax highlighting.
-   **Glassmorphism**: Modern glass effects on cards and navigation.
-   **Responsive Navigation**: Sidebar navigation with scroll-spy and active state highlighting.
-   **Direct Contact**: Simplified "Contact Card" with one-click email functionality.

## üì¶ Getting Started

    ```bash
    git clone https://github.com/Brysonflowers/Bryson-Portfoilo.git
    ```


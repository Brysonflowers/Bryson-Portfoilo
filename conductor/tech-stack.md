# Technology Stack

## Core Frontend
- **React 19:** The primary UI framework, utilizing functional components and hooks for modern state management.
- **TypeScript:** Providing robust type safety and enhancing developer productivity across the codebase.
- **Vite:** Serving as the fast, modern build tool and development server.

## Styling & UI
- **CSS:** Clean and elegant CSS for styling, focusing on modern techniques for a professional "Corporate" aesthetic.

## Development & Tooling
- **ESLint:** Configured with React and TypeScript plugins to ensure high code quality and consistency.
- **Node.js:** The underlying runtime environment for development tools.

## Infrastructure & Deployment
- **GitHub Pages:** Used for hosting the portfolio, with automated deployment using the `gh-pages` package.
- **Git:** Version control for project management and collaboration.
